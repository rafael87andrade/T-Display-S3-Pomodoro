# T-Display-S3-YT
---

### ❗Copy all the `libraries` to the project's `lib` folder.

```
Project/
├── lib/
│   ├── lv_conf.h
│   ├── lvgl/
│   ├── TFT_eSPI/
│   └── ui/
├── src/
│   └── main.cpp
```


>
> To resolve potential build errors, modify your `platformio.ini` file and set:
> ```ini
> platform = espressif32 @ 6.5.0
> ```
